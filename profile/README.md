## 🚀 Overview
Ventoy for Windows is an open-source tool that turns an ordinary USB stick into a flexible multiboot drive. Instead of writing one image at a time and reformatting for every new system, you install Ventoy to the USB once and then simply copy your ISO, WIM, IMG, VHD or EFI files onto it — Ventoy shows a boot menu and launches whichever image you pick. It supports Legacy BIOS and UEFI, works with hundreds of Linux distributions and Windows installation images, and includes Secure Boot support so modern machines can boot your files without disabling firmware protection. For anyone who regularly installs operating systems or runs live environments, a Ventoy USB replaces a drawer full of single-purpose sticks with one drive you keep up to date by drag and drop.

## ✨ Features
- **Copy-and-boot workflow** — install Ventoy to the USB once, then just copy ISO/WIM/IMG/VHD/EFI files onto the drive and boot them from an automatic menu; no repeated formatting.
- **Multiboot from one USB** — keep many operating systems and live environments on a single Ventoy USB and choose which one to launch at startup.
- **Secure Boot support** — boot your images on modern UEFI systems with Ventoy Secure Boot enabled, so you don't have to turn off firmware protection.
- **BIOS and UEFI ready** — works with both Legacy BIOS and UEFI, and handles large image files thanks to exFAT/NTFS support on the USB.
- **Broad image compatibility** — tested with a wide range of Linux distributions, Windows setup media and utility ISOs, making it a versatile alternative when comparing Ventoy vs Rufus for everyday USB creation.

## 🎯 Getting Started
1. Open Ventoy on your Windows PC and plug in the USB drive you want to use.
2. Select the correct USB drive in Ventoy and click **Install** (this erases the drive, so back up any files first).
3. Once installation finishes, open the Ventoy data partition in File Explorer.
4. Copy one or more ISO, WIM, IMG, VHD or EFI files directly onto the USB drive.
5. Restart your computer, boot from the USB, and pick an image from the Ventoy menu to launch it.

## ❓ FAQ
**Is Ventoy free?**
Yes. Ventoy is free, open-source software, and its source code is publicly available on GitHub. You can download and use it at no cost.

**Which versions of Windows are supported?**
Ventoy runs on modern 64-bit Windows, from Windows 7 through Windows 11. For the smoothest experience, use a currently supported Windows version.

**Is Ventoy safe to use?**
Ventoy is an open-source project, so its code can be reviewed by anyone, and Secure Boot support lets you boot images on protected UEFI systems. Keep in mind that installing Ventoy erases the target USB drive, so back up any important files before you start.

**How is Ventoy different from Rufus?**
With Rufus you typically write one image to a USB and reformat to change it, whereas Ventoy installs to the USB once and then boots any ISO you copy onto it. If you frequently switch between multiple images, the Ventoy copy-and-boot approach is usually faster.

**Is my data secure?**
Ventoy works locally on your own machine and does not require an account to create a bootable USB. Because preparing the drive removes existing data, always back up the USB contents first; files you add afterward simply live on the drive as normal.
